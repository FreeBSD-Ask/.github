## 社区失信名单

社区失信名单是在社区工作中出现问题的成员记录，包括不限于失联找不到人，造成社区损失。**极其严重者拉入黑名单，不再列入失信名单。**

被拉入社区失信名单者不能在社区担任任何职务，不能再参与除捐献外任何工作。
